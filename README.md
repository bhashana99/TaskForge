# TaskForge

TaskForge is a scalable backend REST API designed for efficient task and to-do management.  
The project focuses on clean architecture, modular design, and maintainability, providing a strong foundation for future feature expansion.

---

## Project Overview

TaskForge was developed as a backend-focused system to demonstrate:

- Modular API development using NestJS  
- Database integration with MySQL and TypeORM  
- Scalable project structure for enterprise applications  
- Future extensibility for authentication, filtering, and task workflows  

This project highlights best practices in backend engineering and API design.

---

## Key Features

- RESTful API for task management  
- Modular and maintainable backend structure  
- MySQL database integration using TypeORM  
- Clean foundation for future enhancements such as:  
  - User authentication  
  - Task filtering and sorting  
  - Role-based access control  

---

## Tech Stack

- **Backend Framework:** NestJS  
- **Language:** TypeScript  
- **Runtime:** Node.js  
- **Database:** MySQL  
- **ORM:** TypeORM  
- **API Testing:** Postman
